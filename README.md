# Data Extraction & Load Tool (Mini ETL)

A simple Streamlit application that:
- Uploads CSV/Excel/JSON files
- Validates and reads the data using pandas
- Loads the data into a MySQL database using SQLAlchemy
- Shows a preview and logs the process

## Tech Stack
- Python
- Streamlit
- Pandas
- SQLAlchemy
- MySQL

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
